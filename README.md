# AI Resume Analyzer (ATS Score Predictor)

## Problem Statement
Many candidates get rejected due to poor resume optimization for Applicant Tracking Systems (ATS). 
This project analyzes a resume against a job description and provides an ATS match score along with missing skills.

## Features
- Upload Resume (PDF)
- Paste Job Description
- ATS Match Score
- Missing Skills Detection
- Improvement Suggestions
